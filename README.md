📘 Machine Learning Basics

This repository contains short and clear notes on the fundamental concepts of Machine Learning. It is designed for beginners who want a quick and structured introduction to ML concepts.

📖 What is Machine Learning?

Machine Learning (ML) is a branch of Artificial Intelligence that allows systems to learn from data instead of being explicitly programmed. Models identify patterns in data and improve their performance as they are exposed to more data.

🔍 Types of Machine Learning
1. Supervised Learning
Learns from labeled data
Used for prediction tasks
Examples: Classification, Regression
2. Unsupervised Learning
Works with unlabeled data
Finds hidden patterns or structures
Examples: Clustering, Anomaly Detection
3. Reinforcement Learning
Learns through interaction with an environment
Uses rewards and penalties
Examples: Game AI, Robotics
🧠 Key Concepts
Features → Input variables used for prediction
Label/Target → Output variable
Model → Mathematical representation of patterns
Training → Process of learning from data
⚙️ Common Algorithms
Linear Regression
Logistic Regression
Decision Trees
K-Nearest Neighbors (KNN)
Support Vector Machines (SVM)
⚠️ Overfitting vs Underfitting
Overfitting: Model learns noise → poor generalization
Underfitting: Model too simple → misses patterns
📊 Model Evaluation
Accuracy → Correct predictions / total predictions
Precision & Recall → Important for imbalanced data
Train/Test Split → Evaluates performance on unseen data
